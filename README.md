## 一、web机器学习
### 1、web框架
- web框架是表示一组库和模块的开发框架，用来支持动态网站、网络应用程序及网络服务的开发。
### 2、Flask
- Flask是一个基于python的，微型web框架。之所以被称为微型是因为其核心非常简单，同时具有很强的扩展能力。
